# Sign Language AI Roadmap

A visual roadmap tracker for building a Sign Language AI project from scratch to production, integrated with Firebase for progress tracking.

## Quick Start

```bash
npm install
npm run dev
```

## Netlify Deployment
- Connect this repo to Netlify
- Set environment variables:
  - `VITE_FIREBASE_CONFIG` (Firebase config JSON)
  - `VITE_APP_ID` (your app id)
- Netlify will auto-build and deploy from `dist/`
